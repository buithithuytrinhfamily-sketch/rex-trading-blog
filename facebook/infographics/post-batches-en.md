### Comments (mỗi hình 1 comment)

```
🕯️ 1) ANATOMY OF A CANDLE
• Body = open → close price
• Wick (tail) = the highest / lowest price it touched
• Green = up 🟢 | Red = down 🔴

Learn this and you can read ANY chart.
```
```
🟢🔴 2) ENGULFING
When a BIG candle swallows the small one before it, control just changed hands:
• Big green swallows red → BUYERS win
• Big red swallows green → SELLERS win

Strongest at support / resistance.
```
```
🚫 3) THE PIN BAR
A long tail = a rejection.
• Long tail below → buyers rejected low prices → likely UP
• Long tail above → sellers rejected high prices → likely DOWN
```
```
🤔 4) THE DOJI
Open ≈ close → the market is undecided.
Don't rush in — WAIT for the next candle to confirm.
Patience beats guessing 🙂
```
